****************************************************************
matchms: Vector representation of MS data and similarity measure
****************************************************************

`matchms`_ is a Python package that reprenets mass spectral data (e.g. MS-MS)
 as vector and provides spectral similarity measures.

Installation:
    ``conda install matchms``

.. _matchms: https://github.com/matchms/matchms
